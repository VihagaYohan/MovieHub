# MovieHub

## Prerequisites
- Node.js version 12 or later
- npm or yarn
- React native CLI

## Instructions
- Make sure you already setup React native development environment correctly
- Clone the repository and CD to MovieHub project root folder:
- npm install and for iOS move to iOS folder and run pod install

## Running the project
- Start the React Native development server: npm start or yarn start
- In another terminal window, start the emulator or connect your physical device
- Run the app on the emulator or physical device: npx react-native run-android or npx react-native run-ios

## Running the project
If you encounter any issues running the project, try the following steps:
- Make sure you have installed all dependencies by running npm install or yarn install.
- Clear the cache by running npm cache clean or yarn cache clean.
- Delete the node_modules folder and run npm install or yarn install again.
- Check that you have the correct version of Node.js installed by running node -v.
